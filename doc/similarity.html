similarity(options)

new similarity(options)

sim-hash distance越大，相似度越低；similarity percent越大，相似度越高。
Parameters:
Name Type Description
options object 配置项
Properties
Name Type Attributes Default Description
caseSensitive boolean <optional>
 false 可选，false：不区分大小写，true：区分大小写
longTextLength int <optional>
 300 定义长文本的长度界限：其中一个字符串超过此长度时使用sim-hash，否则使用jaro-winkler。
distanceNotPass int <optional>
 3 用于sim-hash，当sim-hash distance小于等于此值时，认为文本不相同；否则认为文本相同
similarityPass float <optional>
 0.7 用于jaro-winkler，当similarity percent大于等于此值时认为文本相同，否则认为文本不同
