similarity(options)

new similarity(options)

sim-hash distance越大，相似度越低；similarity percent越大，相似度越高。
Parameters:
Name Type Description
options object 配置项
Properties
Name Type Attributes Default Description
caseSensitive boolean <optional>
 false 大小写敏感：false：不区分大小写，true：区分大小写
longTextLength int <optional>
 300 定义长文本的长度阈值：其中一个字符串超过此长度时使用sim-hash，否则使用jaro-winkler。
distanceNotPass int <optional>
 3 长文本相似判断阈值[0,Infinity)：当sim-hash distance小于等于此值时，认为文本不相同；否则认为文本相同
similarityPass float <optional>
 0.7 短文本相似判断阈值[0,1]：当similarity percent大于等于此值时认为文本相同，否则认为文本不同
Source:
  • index.js, line 29
Example
const Sim = require('similarity')
const sim = new Sim({
caseSensitive: true,// default:false
longTextLength:100,// default:300
distanceNotPass:2,// default:3
similarityPass:0.6//default:0.7
})
let target = 'abc'
let input = 'Abc'
let res=sim.isSimilarity(target, input)
console.log(res)// false
